# 09 Complex Prompts from Scratch

## 核心理念
- 复杂任务要把前面学过的技巧组合起来用。
- 可以先把角色、规则、示例、步骤、格式都写全，再逐步删减。

## 教程内容
- 推荐骨架包括：上下文、语气、规则、示例、输入、任务、思考、输出格式、prefill。
- 示例 1 是职业顾问聊天机器人。
- 示例 2 是法律问答，要求先找引用再回答。
- 不是每次都要用全套，但复杂场景先堆满更稳。
- 元素顺序会影响效果，可按场景调整。

## 练习内容
- `Exercise 9.1 - Financial Services Chatbot`：写一个能结合税法资料回答 83(b) election 的复杂 prompt。
- `Exercise 9.2 - Codebot`：写一个教学型 prompt，指导修复会除以 0 的 Python 代码。
