# 10.2 Tool Use

## 核心理念
- 工具调用本质上是“模型提请求，外部代码执行，再把结果喂回去”。
- Claude 不会真的执行函数，它只负责产出工具名和参数。

## 教程内容
- 先在系统提示里定义工具规则和工具清单。
- 用 `<function_calls>` / `<invoke>` 格式表示调用请求。
- 外部代码解析参数并真正执行函数。
- 执行结果要按约定格式回传给 Claude。
- 不需要工具时，应直接回答而不是乱调用。

## 练习内容
- `Exercise 10.2.1 - SQL`：为一个小型 users/products 数据集写工具定义，并让模型正确选函数。
