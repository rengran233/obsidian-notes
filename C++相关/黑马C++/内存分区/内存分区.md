代码区：
机器指令

全局区：
全局变量（**而非**局部变量）、静态变量（static）、常量（字符串常量“xxx”、const修饰的全局变量，const修饰的局部变量**不在**全局区）
//执行程序前

栈区：
数据由编译器管理。
如果返回局部变量的地址，数据会被释放。(释放顺序为**先进后出**)
```
int * func(){
	int a = 10;
	return &a;
}
int main(){
	int * p = func();
	cout << \*p << endl; //数据被暂时保留，输出10
	cout << \*p << endl; //输出乱码
}
```
如果使用static变为静态变量，不会被释放



堆区：
由程序员管理
用new字符在堆区开辟数据，也需要程序员手动释放（如析构函数）
