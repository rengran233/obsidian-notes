对于自定义数据类型，需要具体化操作
```
#include <iostream>
using namespace std;

class Person
{
public:
	string m_Name;
	int m_Age;
	Person(string n, int a)
	{
		m_Name = n;
		m_Age = a;
	}
};

template <class T>
bool compare(T& a, T& b)
{
	if (a == b)
		return true;
	else
		return false;
}

template<> bool compare(Person& a, Person& b) //显式具体化
{
	if (a.m_Age == b.m_Age && a.m_Name == b.m_Name)
		return true;
	else
		return false;
}

int main()
{
	Person p1("awa", 10);
	Person p2("awa", 10);
	if (compare(p1, p2))
		cout << "==" << endl;
	else
		cout << "!=" << endl;
	return 0;
}
```


隐式实例化、显式实例化、显式具体化统称为具体化
隐式实例化：编译器根据参数自行推导并生成函数版本
显式实例化：编译器生成指定类型的函数版本
显示具体化：编译器使用为特定类型提供的独立定义
