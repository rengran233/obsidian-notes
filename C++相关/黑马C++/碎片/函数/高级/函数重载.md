1、名称相同
2、作用域相同
3、参数类型、个数或顺序不同
```
void func();
void func(int a, double b);
void func(double a, int b);
```

仅函数返回值类型不同不能重载
```
void func();
int func();
```

引用和常量引用可以作为重载条件
```
void func(int &a);
void func(const int &a);
int main(){
	int a = 10;
	func(a); //调用上
	func(10); //调用下
}
```
int &a = 10不合法，const int &a = 10合法

默认参数可能导致不能重载
```
void func(int a);
void func(int a, int b = 10); //语句无误
int main(){
	func(10); //不可调用
	func(10, 20); //可调用
}
```
