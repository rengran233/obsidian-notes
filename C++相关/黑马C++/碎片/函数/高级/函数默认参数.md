对形参初始化，如果不传入数据，可使用默认值；传入则使用传入值。
```
int func(int a, int b = 10){
	return a + b;
}
int main(){
	cout << func(10) << endl;
	cout << func(10, 20) << endl;
}
```

有默认参数的形参后的形参都必须有默认参数
```
int func(int a, int b = 10, int c) //不可以
```

声明函数与实现函数只能有一个有默认参数
```
int func1(int a = 10, int b = 20)
int main(){}
int func2(int a = 10, int b = 20) //不可行
```
