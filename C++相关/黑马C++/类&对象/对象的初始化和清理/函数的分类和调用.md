构造函数：
分类：有参函数和无参（默认）函数，普通函数和拷贝函数
```
class Person
{
	Person()
	Person(int a)
	Person(const Person &p) //需要是常量且引用
};
int main()
{
	//括号法
	Person p1; //如果用Person p1()会被视为声明函数，而非默认构造函数调用
	Person p2(10);
	Person p3(p2);
	//显示法
	Person p1
	Person p2 = Person(10) //Person(10)为匿名对象，本行执行结束后回收
	Person p3 = Person(p2) //Person(p2)匿名对象
	//不要用匿名对象初始化Person(p3)，会重定义p3
	//隐式转换
	Person p4 = 10 //等于Person p4 = Person(10)
}
```
