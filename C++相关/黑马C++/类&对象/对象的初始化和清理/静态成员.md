关键字static
1.所有类成员共用同一份数据，一改均改
2.编译阶段就分配内存
3.**类**内声明，类外初始化。因为**静态成员属于类本身**，需在类外分配独立存储空间。（不在类内的静态变量不受此限制）
```
class Person
{
public: //静态成员变量也有访问权限
	static int m_A;
};

int Person::m_A = 100; //不表明作用域则为全局变量

int main()
{
	Person p1;
	cout << p1.m_A << endl;
	Person p2;
	p2.m_A = 200;
	cout << p1.m_A << endl; //仍用p1访问
}
```

静态成员变量有两种访问方式
```
class Person
{
public:
	static int m_A;
};

int Person::m_A = 100;

int main()
{
	Person p;
	cout << p.m_A << endl;
	cout << Person::m_A << endl;
}
```

静态成员函数**只能访问静态成员变量**（静态成员函数属于整个类，没有`this`指针，而对于非静态成员的操作需要**具体的对象实例**，所以只能通过向函数传递对象作为参数才能访问该对象的成员变量）
**常用于操作静态成员变量或工具函数**
```
class Person
{
public:
	static int m_A;
	int m_B = 200;
	static void func1()
	{
		cout << m_A << endl;
		//cout << m_b << endl; 不可访问
	}
	static void func2(Person& p)
	{
		cout << p.m_B << endl;
	}
};

int Person::m_A = 100;

int main()
{
	Person p;
	p.func1(); //成员函数同样有两种访问方式
	Person::func2(p);
}
```


```
#include <iostream>
using namespace std;

class base
{
public:
	static int m_A;
	base()
	{
		m_A = 10;
	}
};
int base::m_A = 20;

int main()
{
	cout << base::m_A << endl; //通过类名访问，20
	cout << base().m_A << endl; //创建对象，构造函数修改m_A为10
	cout << base::m_A << endl; //10
	return 0;
}
```
