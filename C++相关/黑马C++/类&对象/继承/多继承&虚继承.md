*不推荐使用*
一个子类继承多个父类
`class <子类> :[继承方式] <父类1>, [继承方式] <父类2>...`


若不同父类中有同名成员，需要用作用域区分（二义性）
```cpp
#include <iostream>
using namespace std;

class Base1
{
public:
	int m_A = 10;
};

class Base2
{
public:
	int m_A = 20;
};

class Son :public Base1, public Base2
{
public:
	int m_B = 30;
	int m_C = 40;
};

void test01()
{
	cout << sizeof(Son) << endl; //4个int数据
	cout << Son().Base1::m_A << endl;
	cout << Son().Base2::m_A << endl;
}

int main()
{
	test01();
	return 0;
}
```


菱形继承问题：
A -> B1, B2 -> C
B1、B2为A的派生类，C多继承B1、B2，此时C中有两份A的属性，但只需要其中一份。
可以通过虚继承解决
```cpp
#include <iostream>
using namespace std;

class Animal 
{
public:
	int m_A;
};

class Sheep :virtual public Animal{}; //virtual虚继承

class Tuo :virtual public Animal{}; //Animal为虚基类

class SheepTuo :public Sheep, public Tuo{};

void test01()
{
	SheepTuo st;
	st.Sheep::m_A = 10;
	st.Tuo::m_A = 20; //，指向同一份数据，更改10为20
	cout << st.Sheep::m_A << endl;
	cout << st.Tuo::m_A << endl;
	cout << st.m_A << endl; //均指向同一份数据
}

int main()
{
	test01();
	return 0;
}
```
原理：创建虚基类指针vbptr(virtual base pointer)，虚基类表vbtable
`Sheep`与`Tuo`中虚基类指针均指向同一份数据
