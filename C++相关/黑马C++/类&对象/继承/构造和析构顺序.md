继承中调用构造函数的顺序：
1.按照**基类**被继承时声明的顺序（从左向右）执行基类的构造函数
2.按照**内嵌成员**对象在类中声明的顺序执行它们的构造函数
3.执行**派生类**的构造函数函数体中的内容
析构函数反之
```cpp
#include <iostream>
using namespace std;

class A
{
public:
	A()
	{
		cout << "A构造函数调用" << endl;
	}
	~A()
	{
		cout << "A析构函数调用" << endl;
	}
};

class B :public A
{
public:
	B()
	{
		cout << "B构造函数调用" << endl;
	}
	~B()
	{
		cout << "B析构函数调用" << endl;
	}
};

int main()
{
	B b;
	return 0;
}
```
A构造函数调用
B构造函数调用
B析构函数调用
A析构函数调用
