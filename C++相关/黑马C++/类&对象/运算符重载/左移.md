通常不用成员函数实现左移运算符重载，若重载为成员函数，则`<<`左操作数为自定义类对象，而不是**流对象**
```cpp
ostream& operator<<(ostream &cout) {};
p.operator<< cout //等价于p << cout，不符合cout在左的习惯
```

使用全局函数
```cpp
#include <iostream>
using namespace std;
class Person
{
public:
	int m_A = 10;
	int m_B = 20;
};
ostream& operator<<(ostream& cout, Person& p) //cout是标准输出流对象
{
	cout << "m_A = " << p.m_A << " m_B = " << p.m_B;
	return cout; //返回仍是cout，可追加
//可以不用cout，因为通过引用传递，相当于起了别名
}
int main()
{
	Person p;
	cout << p << endl; //链式编程
}
```
