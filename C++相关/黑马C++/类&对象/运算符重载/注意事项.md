### **可以改变的**：
1. **运算符的功能**：针对自定义类型（类或结构体），定义运算符的特定行为。  
   ```cpp
   Vector operator+(const Vector& a, const Vector& b) {
       return Vector(a.x + b.x, a.y + b.y);
   }
   ```
2. **运算符的操作数类型**：允许操作数是自定义类型或混合类型。  
   ```cpp
   Complex operator*(const Complex& c, double scalar) { ... }
   ```

---

### **不能改变的**：
1. **运算符的优先级和结合性**：例如，`+`的优先级始终低于`*`，无法通过重载调整。
2. **运算符的操作数个数**：例如，`++`只能是一元运算符，`+`只能是二元运算符。
3. **对基本类型的运算符行为**：如不能重载`int + int`的默认行为。
4. **某些运算符不可重载**：  
   - 作用域解析符 `::`
   - 成员访问符 `.`
   - 条件运算符 `?:`（三元运算符）
   - `sizeof`、`typeid`等类型相关运算符。

---

### **示例：合法与非法**  
✅ 合法：
```cpp
class Matrix {
public:
    Matrix operator*(const Matrix& other) { ... } // 重载*
};
```
❌ 非法：
```cpp
int operator+(int a, int b) { return a - b; } // 不能重载基本类型的运算符
```


- 必须以成员函数方式重载的运算符有 `=` `[]` `()` `->`
- 必须以友元方式重载的运算符有 `<<` `>>`
