### **一、基本概念**
- **功能**：将类的对象隐式或显式转换为基本类型或其他类类型
- **定义格式**：
  ```cpp
  operator 目标类型() const {
      // 转换逻辑
      return 结果值;
  }
  ```

---

### **二、经典示例**
```cpp
class Temperature {
private:
    double celsius;
public:
    Temperature(double c) : celsius(c) {}
    
    // 转换为华氏温度（基本类型）
    operator double() const {
        return celsius * 1.8 + 32;
    }
    
    // 转换为字符串描述（其他类类型）
    operator std::string() const {
        return std::to_string(celsius) + "°C";
    }
};

int main() {
    Temperature t(25.0);
    double fahrenheit = t;   // 触发operator double()
    std::string desc = t;    // 触发operator std::string()
}
```

---

### **三、重要特性**
1. **必须是成员函数**，不能是全局函数
2. **没有返回类型声明**（返回类型由运算符名决定）
3.  **无参数**：类型转换运算符不接收任何参数
4. **通常为const成员函数**：保证转换过程不修改对象状态
5. **可以声明为explicit**（C++11+）：禁止隐式转换，需显式调用
   ```cpp
   explicit operator bool() const { 
       return !empty(); 
   }
   ```

---

### **四、应用场景**
1. 数值类转换（如分数转小数）
2. 智能指针转原始指针
3. 自定义字符串类转std::string
4. 状态判断（如文件流转bool判断是否打开成功）
