`""`双引号定义**字符串**字面量，`''`单引号定义单个**字符**字面量
`""`是一个合法字符串，表示空，不含任何字符。
`""`中字符串含有隐藏的`\0`结束空字符，如`"awa"`占用4字节
`' '`空格字符，`'\0'`空字符/终止符，`cout`、`strlen()`遇到`'\0'`会终止读取，`sizeof()`不会


空格`' '`：32
`'0' - '9'`：48-57
`'A' - 'Z'`：65-90
`'a' - 'z'`：97-122




😊 正确答案是 **D. A和C皆有可能**，详细解析如下：

---

### **核心分析**
1. **选项A**：  
   - 若类 `A` 定义了**无参构造函数**（默认构造函数或用户显式定义的无参构造函数），则创建数组时会调用该构造函数10次。  
   - 示例：  
     ```cpp
     class A {
     public:
         A() {} // 无参构造函数
     };
     ```

2. **选项C**：  
   - 若类 `A` 的构造函数**所有参数都有默认值**，则它等效于无参构造函数，可以在数组创建时被调用10次。  
   - 示例：  
     ```cpp
     class A {
     public:
         A(int x = 0) {} // 参数有默认值
     };
     ```

3. **排除选项B**：  
   - 拷贝构造函数用于对象复制（如 `A a2(a1)`），而 `new A[10]` 是构造新对象，与拷贝构造函数无关。

---

### **结论**
动态数组 `new A[N]` 要求类必须提供**可无参调用的构造函数**，这包括显式无参构造函数（A）或全默认参数构造函数（C）。因此，选项D最全面~ 🚀
