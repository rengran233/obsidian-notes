常成员函数是指在函数的声明和定义的末尾加上`const`关键字的成员函数。它表示这个函数**不会修改对象的任何成员变量**（除了mutable修饰的成员变量）。

常函数**可以访问所有成员变量**，但不能修改，除非使用`mutable`声明变量；但**不能访问不用`const`修饰的成员函数**，因为其可能修改对象


常对象是指在声明对象时加上`const`关键字的对象。它表示这个对象的**所有成员变量都是常量，不能被修改**。

**常对象只能调用常成员函数**，因为常成员函数保证不会修改对象的状态。非常成员函数可能会修改对象的状态，所以不能被常对象调用。
```cpp
class myClass
{
public:
	void func() { cout << "call func()"; }
	void const_func() const { cout << "call const_func()"; }
};

int main()
{
	myClass const const_obj;
	//const_obj.func(); //不可调用
	const_obj.const_func();

	return 0;
}
```

常数据成员（`const`修饰的成员变量）**必须在构造函数的初始化列表中初始化**，不能在构造函数体内**赋值**（函数体内操作本质是赋值而非初始化），且初始化后不可修改。在C++11后可以在声明时直接初始化
