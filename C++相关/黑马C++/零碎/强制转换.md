```
char a;
cout << a - 32 //输出为int
cout << char(a - 32) //输出为char (即A)

int a = 3, b = 2;
cout << double(a) / b << '\n';
cout << (double)a / b << '\n'; //C风格强制转换
```


C++风格强制转换：
```
//static_cast静态转换
double d = 3.14;
int i = static_cast<int>(d);  // 3

Base* base = new Derived();
Derived* derived = static_cast<Derived*>(base);  // 不安全！需确保类型正确

//dynamic_cast动态转换
//const_cast常量性转换
//reinterpret_cast重新解释转换
```


字符'0'和数字0之间ASCII相差48，`char(48)` == `0`，其他数字也是如此
同时`'9' - '0'` == `9`
大写字母和小写字母之间相差32，`'A' + 32` == `'a'`
同时`'B' + 'a' - 'A'` == `'b'`
