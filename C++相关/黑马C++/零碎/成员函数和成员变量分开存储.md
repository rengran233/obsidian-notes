只有非静态成员变量属于类的对象上
```
class Person {};

int main()
{
	cout << sizeof(Person); //输出为1
}
```
空对象占用1字节，区分

```
class Person
{
	int m_A;
};

int main()
{
	cout << sizeof(Person); //输出为4
}
```
int类型占用4字节

```
class Person
{
	int m_A;
	static int m_B;
	void func1();
	static void func2();
};

int main()
{
	cout << sizeof(Person);
}
```
静态成员变量与成员函数均不属于类的对象上
