**指针 ± 整数**：
调整指针指向的位置（偏移量由类型大小决定）。
`p + 1`指向该数组下一个元素，`p - 1`同理

**指针减法：**
指针的减法运算结果表示的是两个指针之间相隔的**元素个数**（而非字节数）。其计算方式是**字节差除以元素类型的大小**，并且结果会被**自动取整**（向零截断）
```cpp
char s[50] = "The Object-Oriented Programming Language!";
short *p1 = (short*)(s+1);  // 将s的地址偏移1字节后转为short*
short *p2 = (short*)(s+17); // 偏移17字节后转为short*
//p1和p2被强制转换为short*，需确保转换后的地址在数组范围内（否则可能越界）。
```
char数据类型占用1字节，s+1令指针偏移1字节。所以p2和p1间隔16字节，short数据类型占用2字节，则p2 - p1为8，即间隔8个元素。
相减的指针**必须指向同一数组或数组末尾之后的位置**

**指针加法：**
指针之间**不能**进行加法操作

**指针比较**：判断指针指向的地址高低（如 `p1 < p2`）。
指针比较的两指针**必须指向同一数组或数组末尾之后的位置**

**指针赋值**：同类型指针间的赋值。
