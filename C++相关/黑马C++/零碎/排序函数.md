`sort()`排序函数包含于`#include<algorithm>`
`sort(begin, end)`, 排序\[begin, end)区间元素
如`sort(a+1, a+4)`排序数组a\[1]到a\[3]三个元素

`sort()`函数默认通过`<`运算符进行排序，可以通过重载`<`对自定义数据类型进行排序

`sort()`排序是不稳定排序，相同元素可能会在排序过程中交换位置，`stable_sort()`可以实现稳定排序

自定义排序规则
```cpp
bool myCompare(int& v1, int& v2)
{
	return v1 > v2;
}
int main()
{
	vector<int> v {1, 3, 5};
	sort(v.begin(), v.end(), myCompare); //降序排列
}
```
自定义数据类型排序
```cpp
class Person
{
public:
	string name;
	int age;
	int height;
	Person(string n, int a, int h) : name(n), age(a), height(h) {};
};

void print(const vector<Person>& v)
{
	for (const Person p : v)
	{
		cout << p.name << ' ' << p.age << ' ' << p.height << '\n';
	}
}

bool myCompare(Person& p1, Person& p2)
{
	if (p1.age == p2.age)
		return p1.height > p2.height; //降序
	return p1.age < p2.age; //升序
}

int main()
{
	vector<Person> v;
	v.push_back(Person("q", 11, 15));
	v.push_back(Person("g", 15, 20));
	v.push_back(Person("h", 18, 11));
	v.push_back(Person("c", 11, 50));
	print(v);
	cout << "--------------------\n";
	sort(v.begin(), v.end(), myCompare);
	print(v);

	return 0;
}
```
