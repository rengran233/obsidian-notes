这是一份针对 React 开发者的 Vue 3 `<script setup>` 语法逐行/逐块详解。我将以你当前打开的 `oc-button/index.vue` 文件为例，将代码拆解并注释说明。

### 1. Template 部分 (视图层)

Vue 的 template 类似于 React 的 JSX，但是它更像标准的 HTML，通过“指令”增强功能。

```html
<template>
  <!-- 
    ref="buttonRef": 
    对应 React 的 ref={buttonRef}。在 script 中可以通过 buttonRef.value 访问该 DOM。
    
    :data-e2e="performBaseKey": 
    冒号 : 是 v-bind: 的缩写。表示属性值是一个变量或表达式。
    React 写法: data-e2e={performBaseKey}
    
    :class="{ ... }":
    动态 class 绑定。Vue 支持对象语法 { '类名': boolean }。
    React 写法: className={classNames({ 'oc-button-wrap': true, ... })}
    
    @mouseenter="handleMouseEnter":
    @ 是 v-on: 的缩写，用于绑定事件。
    React 写法: onMouseEnter={handleMouseEnter}
  -->
  <div
    ref="buttonRef"
    :class="{
      [`${ocCssPrefix}oc-button-wrap`]: true, 
      [`${ocCssPrefix}oc-button-selected-wrap`]: selected, // props.selected 为 true 时应用该类
    }"
    :style="style" 
    @mouseenter="handleMouseEnter"
  >
    <!-- 使用引入的组件 OButton -->
    <o-button
      :class="buttonClass"
      :size="size" 
      @click="onButtonClick" 
    >
      <!-- 
        v-if="prefixIcon":
        条件渲染。如果不满足，DOM 根本不会生成。
        React 写法: { prefixIcon && <oc-icon ... /> }
      -->
      <template v-if="prefixIcon">
        <oc-icon :config="prefixIcon"></oc-icon>
      </template>

      <!-- 
        {{ text }}:
        文本插值。
        React 写法: {text}
      -->
      <span v-if="text">
        {{ text }}
      </span>

      <!-- 
        <slot></slot>:
        插槽，用于渲染父组件传递的子元素。
        React 写法: {children} 或 {props.children}
      -->
      <slot></slot>

      <!-- 
        v-bind="popoverOption":
        批量绑定属性。
        React 写法: {...popoverOption}
      -->
      <template v-if="showPop">
        <oc-popover
          v-bind="popoverOption"
          @imbot="(val: string) => $emit('imbot', val)" 
        />
      </template>
    </o-button>
  </div>
</template>
```

### 2. Script 部分 (逻辑层)

这是 Vue 3 的 **Composition API** (`<script setup>`)，逻辑组织方式非常像 React Hooks。

```typescript
// lang="ts" 启用 TypeScript, setup 启用 Composition API 语法糖
<script lang="ts" setup>
// 引入 Vue 的核心 Hook
import { computed, ref, defineSlots, onMounted, onUnmounted } from "vue";
// 引入类型定义
import { IOcButtonProps, IOcButtonEmits, IOcButtonSlots } from "./type";

// 1. Props 定义
// defineProps 是编译器宏，不需要 import。
// withDefaults 用于给 Props 设置默认值。
// React 对应: function Button({ size = 'md', ...props }: Props)
const props = withDefaults(defineProps<IOcButtonProps>(), {
  size: "md",
  disabled: false,
  // ... 其他默认值
});

// 2. Emits 定义 (自定义事件)
// Vue 组件触发父组件回调是通过 emit('eventName', payload)
// React 对应: props.onClick(e)
const emit = defineEmits<IOcButtonEmits>();

// 3. Slots 类型定义 (仅用于 TS 类型检查)
defineSlots<IOcButtonSlots>();

// 4. 响应式状态 (State)
// ref 用于定义基本类型的响应式数据。
// 注意: 在 script 中读写需要用 showPop.value，在 template 中直接用 showPop。
// React 对应: const [showPop, setShowPop] = useState(false);
const showPop = ref(false);
const buttonRef = ref(null); // 对应 template 里的 ref="buttonRef"

// 5. 计算属性 (Computed)
// 自动追踪依赖，只有依赖变化时才重新计算。
// React 对应: const disabledStatus = useMemo(() => props.disabled || props.loading, [props.disabled, props.loading]);
const disabledStatus = computed(() => props.disabled || props.loading);

// 6. 方法 (Methods)
// 直接定义函数即可在 template 中使用。
const handleMouseEnter = () => showPop.value = !!popoverMsg.value;

const onButtonClick = (e: Event) => {
  if (disabledStatus.value) { // 访问 ref/computed 值要加 .value
    return;
  }
  // 触发名为 'click' 的事件，父组件通过 @click="..." 监听
  emit("click", e); 
};

// 7. 生命周期 (Lifecycle)
// React 对应: useEffect(() => { ... return () => { ... } }, [])
onMounted(() => {
  // 组件挂载完成
  performColFnStatus(ActionEnum.ComponentMounted);
});

onUnmounted(() => {
  // 组件卸载
  showPop.value = false;
});
</script>
```

### 3. Style 部分 (样式层)

```less
// lang="less" 指定预处理器
<style lang="less">
@import "../style/variables.less";

// 这里的样式默认是全局的！
// 除非加上 scoped 属性: <style scoped>
// 但这个项目似乎使用 BEM 命名规范 (oc-button-wrap) 来避免冲突，而不是 Scoped CSS。

.@{oc-css-prefix}oc-button-wrap {
  display: inline-flex;
  // Less 嵌套语法
  &-block {
    width: 100%;
  }
}
</style>
```

### 总结：给 React 开发者的快速映射

*   **`useState`** -> **`ref`** (记得 `.value`)
*   **`useMemo`** -> **`computed`** (无需手动写依赖数组)
*   **`useEffect`** -> **`watch`** / **`watchEffect`**
*   **`useEffect([]`** -> **`onMounted`**
*   **`className`** -> **`:class`**
*   **`{children}`** -> **`<slot />`**
*   **`props.onClick`** -> **`emit('click')`** (配合 `@click`)
