macOS 的快捷键非常丰富，不需要死记硬背。为了帮您快速上手（特别是您正在使用 Windows 键盘），我将它们按**使用逻辑**分成了五大类。

掌握了这些，您就掌握了 macOS 90% 的效率精髓。

> **前置提醒 - 核心修饰键：**
> * **Command (⌘) / Win键：** 它是绝对的主角（相当于 Windows 的 Ctrl）。
> * **Option (⌥) / Alt键：** 它是备选/副角。
> * **Control (⌃) / Ctrl键：** 在 macOS 里是配角。

---

### 一、 每天都在用的“肌肉记忆”

这些键位和 Windows 逻辑非常像，只是把 `Ctrl` 换成了 `Command`。

| 功能          | 快捷键                   | 备注            |
| ----------- | --------------------- | ------------- |
| **复制 / 粘贴** | `⌘ + C` / `⌘ + V`     | 最基础。          |
| **全选**      | `⌘ + A`               |               |
| **撤销 / 重做** | `⌘ + Z` / `⌘ + ⇧ + Z` | 重做多了一个 Shift。 |
| **保存**      | `⌘ + S`               |               |
| **查找**      | `⌘ + F`               | 网页、文档通用。      |
| **打印**      | `⌘ + P`               |               |
| **移动**      | `⌘ + ⌥ + V`           | 在复制的基础上使用     |

### 二、 窗口与软件管理（逻辑与 Windows 不同）

这是新手最容易混淆的地方，请务必区分 **“关闭窗口”** 和 **“退出软件”**。

| 功能 | 快捷键 | 核心解释 |
| --- | --- | --- |
| **退出 App** | `⌘ + Q` | **Quit**。彻底关掉程序（不留后台）。 |
| **关闭窗口** | `⌘ + W` | **Window**。只关掉当前窗口，App 还在后台运行。 |
| **最小化** | `⌘ + M` | **Minimize**。缩到程序坞里。 |
| **隐藏 App** | `⌘ + H` | **Hide**。老板键！瞬间让当前软件消失，桌面变清爽。 |
| **切换 App** | `⌘ + Tab` | 按住 Cmd 不放，按 Tab 切换。 |
| **强制退出** | `⌘ + ⌥ + Esc` | 相当于 Windows 的 `Ctrl+Alt+Del`，专治软件卡死。 |

### 三、 访达（Finder）与文件操作

在桌面上或文件夹里使用。

| 功能 | 快捷键 | 备注 |
| --- | --- | --- |
| **预览文件** | **空格键** | **Mac 神技**。选中文件按空格，直接预览（无需打开）。 |
| **删除文件** | `⌘ + Delete` | Windows 键盘是用 `Win + Backspace`。 |
| **清倒废纸篓** | `⌘ + ⇧ + Delete` | 彻底删除。 |
| **重命名** | **回车 (Enter)** | **注意：** Mac 上回车是重命名，不是打开文件！ |
| **打开文件** | `⌘ + O` 或 `⌘ + ↓` | 这才是打开文件。 |
| **新建文件夹** | `⌘ + ⇧ + N` |  |
| **调出设置** | `⌘ + ,` (逗号) | 几乎所有 App 的设置页面都在这里。 |

### 四、 截图与录屏（非常强大）

Mac 自带的截图功能非常强，不需要第三方软件。

| 功能 | 快捷键 | 说明 |
| --- | --- | --- |
| **全屏截图** | `⌘ + ⇧ + 3` | 咔嚓一声，保存到桌面。 |
| **区域截图** | `⌘ + ⇧ + 4` | 变成十字光标，框选哪里截哪里。 |
| **窗口截图** | `⌘ + ⇧ + 4` 后按 `空格` | 光标变成相机，点击某个窗口，截出带阴影的完美窗口图。 |
| **录屏/选项** | `⌘ + ⇧ + 5` | 调出截图工具栏，可以录制屏幕视频。 |

### 五、 文本编辑与光标移动（效率提升十倍）

写代码或写文档时，手指不想离开键盘去摸鼠标？请记住这组“光标瞬移”大法。

| 想要去哪里 | 快捷键 | 逻辑记忆 |
| --- | --- | --- |
| **跳到行首 / 行尾** | `⌘ + ←` / `→` | Command 代表“绝对”，直接到底。 |
| **跳到文首 / 文末** | `⌘ + ↑` / `↓` | 直接跳到整篇文章的最上面或最下面。 |
| **按单词跳跃** | `⌥ + ←` / `→` | Option 代表“备选/精细”，一个词一个词地跳。 |
| **整行删除** | `⌘ + Delete` | 从光标处直接删到行首（慎用）。 |
| **按单词删除** | `⌥ + Delete` | 从光标处删除前一个单词。 |

### 六、 系统级的“神键”

| 功能 | 快捷键 | 描述 |
| --- | --- | --- |
| **聚焦搜索** | `⌘ + 空格` | **Spotlight**。这是 Mac 的大脑，用来找软件、找文件、算汇率。 |
| **切换输入法** | `⌃ + 空格` | Control + 空格（老用户习惯），或者按 `Fn` (如果键盘支持)。 |
| **显示桌面** | `F11` 或 `⌘ + F3` | 视具体设置而定，通常建议用触控板手势替代。 |
