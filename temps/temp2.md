### 回顾变换公式

  取圆柱半径 $R=1$，观察者距离 $d$，核心映射：

  $$\rho(s,\phi) = \frac{s(d^2-1)}{A(\phi) - s \cdot B(\phi)}, \quad A(\phi)=d^2-2d\cos\phi+1,\; B(\phi)=d-\cos\phi$$

  其中 $(s,\phi)$ 是镜面图案极坐标，$(\rho,\phi)$ 是纸面图案极坐标。方位角 $\phi$ 不变——这是整个分析的关键。

  ## 第二问分析

  ### 问2-Part A：不指定镜面图案，能否让两者都有意义？

  #### 自由度分析

   变量                    │ 自由度
  ─────────────────────────┼────────────
   镜面图案 $I_m$          │ $N$ 个像素
   纸面图案 $I_p$          │ $N$ 个像素
   变换约束 $I_p = T(I_m)$ │ $N$ 个方程
   参数 $(R,d,h)$          │ 3 个

  约束后剩余自由度：$N + 3 - N = 3$。只指定变换关系不指定图案时，有无穷多解，3个几何参数就是剩余自由度。

  所以数学上一定可行，关键是怎么找。

  #### 三种入手策略

  策略1：从纸面出发，反推镜面

    选定一个有意义的纸面图案 I_p → 计算 I_m = T^{-1}(I_p) → 检查 I_m 是否有意义

  • 优点：纸面图案我们可以"设计"，比如画一个放射状的太阳、写一圈环形文字
  • 风险：$T^{-1}$ 也是强扭曲变换，反推出的镜面图案可能不可辨认

  策略2：从镜面出发，正推纸面

    选定一个有意义的镜面图案 I_m → 计算 I_p = T(I_m) → 检查 I_p 是否有意义

  • 这正是题目说"目前作品中纸面图案都是无意义的"——说明大多数镜面图案经变换后纸面确实无意义
  • 需要挑选"对变换友好"的镜面图案

  策略3（推荐）：协同设计

  核心思想：先选一个与变换几何特性兼容的图案类型，再微调参数使双方都有意义。

  变换的几何特性决定了什么样的图案"两面都能辨认"：

   变换特性               │ 对图案的要求
  ────────────────────────┼────────────────────────────────────────────────
   方位角 $\phi$ 不变     │ 两个图案的显著特征必须在相同角度方向上
   径向拉伸随 $\phi$ 变化 │ 图案如果是径向对称的，拉伸效果均匀，更容易辨认
   中心有空洞             │ 两个图案中心区域都应该是空白/不重要的
   远端细节被极度拉伸     │ 两个图案的外围应该是简单/模糊的

  具体可操作的图案选择：

  1. 放射线/星芒图案：镜面是星芒→纸面也是星芒（只是间距变了），两个都看得出来
  2. 环形文字：镜面是沿圆周排列的文字→纸面是沿变形圆周排列的文字，文字虽然扭曲但沿圆周排列的特征保留
  3. 同心圆/波纹：镜面是同心圆→纸面是偏心椭圆族，仍可辨认
  4. 花卉/太阳：天然具有径向对称性

  #### 数学判断标准

  定义"可辨认度"指标：

  $$Q(I) = \alpha \cdot S(I) + \beta \cdot H(I) + \gamma \cdot C(I)$$

  • $S(I)$：结构相似性——与已知有意义模板（文字、人脸、动物轮廓）的匹配度
  • $H(I)$：信息熵——不能太低（白噪声）也不能太高（纯噪声）
  • $C(I)$：连通性——有意义区域应连通，不能过于破碎

  优化目标：$\max_{R,d,h,I_m} \min(Q(I_m),\ Q(T(I_m)))$

  ### 问2-Part B：指定镜面图案，纸面图案能否有意义？

  #### 问题本质

  给定 $I_m$，$I_p = T_{R,d,h}(I_m)$ 完全由参数 $(R,d,h)$ 决定。只有3个自由度。

  大多数情况下答案是不行——因为变换把正常图像扭曲成"混乱线条"，无论怎么调参，扭曲的本质不变。

  #### 但有例外

  例外1：镜面图案本身具有与变换兼容的结构

  如果 $I_m$ 恰好在变换后能形成某种可辨认模式。例如：

  • $I_m$ 是一个高度对称的图案，变换后的扭曲恰好形成另一个对称图案
  • $I_m$ 的像素分布恰好使得 $T(I_m)$ 的灰度分布形成可辨认的形状

  例外2：利用参数调优

  $(R,d,h)$ 虽然只有3个参数，但可以显著改变扭曲程度：

  • $d$ 很大（观察者很远）→ 变换趋近于恒等（几乎不扭曲），纸面图案接近镜面图案
  • $d$ 接近 $R$ → 扭曲极大

  所以如果允许 $d$ 很大，纸面图案可以接近镜面图案，自然有意义。但这失去了"反射变形"的艺术意义。

  例外3：对镜面图案做"预处理"

  不是直接用给定的 $I_m$，而是对它做艺术简化：

  • 提取轮廓线，只保留骨架
  • 降低分辨率（像素化）
  • 只保留低频分量

  简化后的 $I_m'$ 经变换后，纸面图案 $T(I_m')$ 可能因为低频特性而更容易辨认。

  #### 数值实验思路

    对指定镜面图案 I_m:
      for d in [2R, 3R, 5R, 10R, 20R]:
        for R in [0.5, 1, 2, 3]:
          I_p = T_{R,d}(I_m)
          score = meaningfulness(I_p)
          记录 (R, d, score)
      选最优参数

  ## 第三问分析

  ### 问题本质

  同时给定 $I_m$（镜面图案）和 $I_p$（纸面图案），能否通过艺术加工使两者都有意义？

  数学上：找 $(R,d,h)$ 及修改量 $(\Delta_m, \Delta_p)$ 使得

  $$|T_{R,d,h}(I_m + \Delta_m) - (I_p + \Delta_p)| < \varepsilon$$

  且 $I_m + \Delta_m$ 和 $I_p + \Delta_p$ 都有意义。

  --------

  ### 核心矛盾：3个参数 vs N个约束

  • 约束数：$N$（像素级匹配）
  • 自由参数：$(R,d,h)$ = 3个 + 艺术修改的自由度

  如果艺术修改不引入额外自由度（只是微调），问题严重欠约束，精确匹配几乎不可能。

  但如果把"有意义"降格为"低分辨率可辨认"，约束数骤降：

  • 一幅 $1000 \times 1000$ 的图有 $10^6$ 个约束
  • 但如果只要求 $10 \times 10$ 分辨率可辨认，只有 $100$ 个约束
  • 加上3个几何参数 + 艺术简化的自由度，就变得可解

  --------

  ### 两个图案必须满足的条件

  #### 条件1（必要）：方位角对齐

  变换保持 $\phi$ 不变，所以：

    镜面图案在方位角 $\phi$ 处的显著特征，必须对应纸面图案在同一方位角 $\phi$ 处的显著特征。

  举例：如果镜面图案在 $\phi=0$（朝向观察者方向）有一个人脸，纸面图案在 $\phi=0$ 也必须有某种可辨认的内容。如果纸面图案在 $\phi=0$ 是空白，就不兼容。

  检验方法：对两个图案分别做径向投影（沿 $\phi$ 方向积分），比较两者的角分布是否相似：

  $$P_m(\phi) = \int_0^{s_{max}} |I_m(s,\phi)|\,ds, \quad P_p(\phi) = \int_{R}^{\rho_{max}} |I_p(\rho,\phi)|\,d\rho$$

  如果 $P_m(\phi)$ 和 $P_p(\phi)$ 的峰值位置基本一致，则满足方位角对齐。

  #### 条件2（必要）：径向单调性兼容

  变换中，对固定 $\phi$，$\rho$ 是 $s$ 的严格单调递增函数（在有效域内）。所以：

    镜面图案中从内到外的径向顺序，必须在纸面图案中保持一致。

  即：如果镜面图案中特征A在特征B的内侧（更靠近圆柱轴），那么纸面图案中A的对应点也必须在B的内侧。

  违反此条件的例子：镜面图案是"内圈文字+外圈图案"，纸面图案是"内圈图案+外圈文字"——不可实现。

  #### 条件3（必要）：中心空洞兼容

    纸面图案在圆柱覆盖区域（$\rho < R$）内不得有重要内容。

  这是因为圆柱本身占据了中心区域，纸面上这个区域被遮挡。

  同时：

    镜面图案在 $s$ 接近0的区域的内容，会映射到纸面上极远处（$\rho \to \infty$），实际不可用。

  所以镜面图案中心也不应有重要内容。

  #### 条件4（充分性方向）：信息密度匹配

  变换的"雅可比行列式"衡量了局部面积放大率：

  $$J(s,\phi) = \frac{\partial \rho}{\partial s} \cdot \frac{\rho}{s} = \frac{(d^2-1)[A(\phi)]}{[A(\phi) - s \cdot B(\phi)]^2} \cdot \frac{\rho}{s}$$

  • $J > 1$：纸面上该区域被放大（镜面一个小区域→纸面一个大区域）
  • $J < 1$：纸面上该区域被压缩

    如果镜面图案在 $J$ 大的区域有丰富细节，纸面上这些细节会被"摊薄"；如果纸面图案在 $J$ 大的区域需要丰富细节，镜面图案必须在对应小区域有更高密度的信息。

  实操意义：两个图案的信息密度分布必须与 $J$ 的分布兼容。不能要求一个"信息密集"的区域对应一个"信息稀疏"的区域。

  --------

  ### 具体求解方法

  #### 方法1：迭代优化法

    输入: 目标 I_m, I_p
    初始化: R, d, h

    repeat:
      1. 计算 I_p' = T_{R,d,h}(I_m)
      2. 计算误差 E = ||I_p' - I_p||
      3. 对 (R,d,h) 做梯度下降，最小化 E
      4. 对 I_m 做允许范围内的艺术修改（简化轮廓、调整灰度等）
      5. 对 I_p 做允许范围内的艺术修改
    until E < ε 或达到迭代上限

  #### 方法2：特征点匹配法

    6. 从 I_m 中提取显著特征点集 F_m = {(s_i, φ_i)}
    7. 从 I_p 中提取显著特征点集 F_p = {(ρ_j, φ_j)}
    8. 按 φ 对齐两组特征点
    9. 对每组 (φ_k, s_i, ρ_j)，验证是否满足变换关系：
       ρ_j ≈ s_i · (d²-R²) / [A(φ_k) - s_i·B(φ_k)]
    10. 用最小二乘拟合 (R, d)
    11. 不能匹配的特征点 → 需要艺术修改

  #### 方法3：频域分析法

    12. 对两个图像做极坐标下的2D傅里叶变换
    13. 由于变换保持方位角，频率的角向分量应该对齐
    14. 径向频率分量需要通过变换的传递函数关联
    15. 不兼容的频率分量 → 需要滤除（艺术简化）

  --------

  ### 总结：三个问题的递进关系

    问题1: I_m → I_p (完全可解，公式直接算)
             自由度: ∞ (I_m 任意)

    问题2A: 找 (I_m, I_p) 使得两者有意义
             自由度: ∞ (都不指定)
             → 可行，关键是选与变换兼容的图案

    问题2B: 给定 I_m，找 I_p 有意义
             自由度: 3 (只有 R,d,h)
             → 大多数情况不行，需要 I_m 本身与变换兼容

    问题3: 给定 (I_m, I_p)，通过艺术加工使两者有意义
             自由度: 3 + 艺术修改量
             → 需要满足4个必要条件，不满足则需要足够的艺术修改空间
             → 条件1(方位角对齐)是最核心的不可绕过约束

  最核心的数学结论：变换保持方位角不变（$\phi_{paper} =
  \phi_{mirror}$），这是所有三个问题中最根本的约束。它意味着两个图案必须有"相同的角方向结构"——无论怎么艺术加工，这一点无法改变。
