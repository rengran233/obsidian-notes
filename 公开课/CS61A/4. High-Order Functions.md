(2) **Function as return value:**
Functions defined within other function bodies are bound to names in a local frame.
```python
def make_adder(k):
    """ return a function takes an arg n and returns n+k
    >>> adder_three = make_adder(3)
    >>> adder_three(4)
    7
    """
    def adder(n):
        return n + k # can refer to names in a enclosing function
    return adder
```

In the form of `lambda <argument>: <statement>`
```python
# Inverse a function
def search(func):
    x = 0
    while not func(x):
        x += 1
    return x

def inverse(origin_function):
	# return a g(y) such that f(g(x)) == x
    return lambda y : search(lambda x: origin_function(x) == y)
```
