### **1.extends syntax**
```Java
// use keyword extends to inheritance a class instead of interface
public class NewSLList extends SLList {
	SLList deletedItems;
	// deletedItems can also be initialized while declared
	public NewSLList() {
		// explicitly call a superclass constructor
		super();
		deletedItems = new SLList();
	}
	// override a new removeLast() to record deleted items
	@Override
	public int removeLast() {
		// use super to access the method of superclass
		int x = super.removeLast();
		deletedItems.addLast(x);
		return x;
	}
} 
```
- `extends` is the keyword for class inheritance
- `super.` can access the method of superclass to avoiding copying and pasting and accessing private member of superclass directly to reuse code
- **Constructors are not inherited**, but every subclass constructor must **start** with a superclass constructor. If we not explicitly call one, Java will automatically call a **non-argument** one.

### **2.the object class**
Every **class**(not interface) in Java extends the `object` class implicitly and it has several methods like `equals()`, `hashCode()`, `toString()`

### **3.Casting**
```Java
public static void main(String[] args) {
   	VengefulSLList<Integer> vsl =  new VengefulSLList<Integer>(9);
   	SLList<Integer> sl = vsl; // VengefulSLList is a SLList
	// work normally
	sl.addLast(50);
   	sl.removeLast(); 
	// compile time error
    sl.printLostItems(); 
    VengefulSLList<Integer> vsl2 = sl;
}
```
- `sl.printLostItems()` will cause a compile time error because `sl`'s static type is SLList and a SLList have no `.printLostItems()` method
- `VengefulSLList<Integer> vsl2 = sl` will cause a compile time error because a subclass type can't "contain" a superclass instance without **casting**

### **4.Higher order functions**
```Java
// IntUnaryFunction.java
public interface IntUnaryFunction {
    int apply(int x);
}
// 
public class TenX implements IntUnaryFunction {
    /* Returns ten times the argument. */
    public int apply(int x) {
        return 10 * x;
    }
}

public static int do_twice(IntUnaryFunction f, int x) {
    return f.apply(f.apply(x));
}

System.out.println(do_twice(new TenX(), 2));
```
