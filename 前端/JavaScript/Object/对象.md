## 1.访问对象

访问 JavaScript 中的对象属性主要有两种方式：点符号`.`和括号符号`[]`。
```js
// 点符号
const person = {
  name: "Alice",
  age: 30,
  city: "New York"
};

console.log(person.name);  // Alice
console.log(person.age);   // 30
// 点符号可以通过有效的JS标识符访问

// 括号符号
const oddObject = {
  "1stProperty": "Hello",
  "property with spaces": "World"
};

console.log(oddObject["1stProperty"]);  // Hello
console.log(oddObject["property with spaces"]);  // World
// 括号要更加灵活，可以通过字符串访问
let propertyName = "property with spaces";
console.log(person[propertyName]); 
// 也可以使用变量动态访问
```


## 2.移除对象

移除对象的方式有两种，分别是**用`delete`操作符移除对象**和通过**解构加剩余参数创建没有特定元素的新对象**
```js
// 删除
const person = {
  name: "Alice",
  job: "Engineer"
};
delete person.job;
console.log(person.job); // undefined

// 剩余
const person = {
  name: "Bob",
  age: 25,
  job: "Designer",
  city: "New York"
};
const { job, city, ...remainingProperties } = person;
console.log(remainingProperties); // { name: "Bob", age: 25 }
```
