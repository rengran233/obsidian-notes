`Number`, `String`, `Boolean`, `Null`, `Undefined`, `NaN`, `Bigint`, `Symbol`

`NaN`本质是`Number`类型，代表应返回数字类型却产生了无法表示或未定义的结果，
```js
console.log(typeof(0/0)); // "number"
console.log(NaN === NaN); // false
```
判断是否是`NaN`类型有两个方法`isNaN()`和`Number.isNaN()`。`isNaN()`只要参数不能转换为数字类型时就返回`true`，而`Number.isNaN()`只有参数为`NaN`类型是才会返回`true`
