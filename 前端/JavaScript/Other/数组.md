js的数组索引为zero-base

数组拥有`array.length`属性返回数组长度

js中数组越界访问会返回`undefined`而非报错，同时支持了直接对未定义的索引进行赋值，但对过大的索引赋值会导致数组中存在大量的未定义项

`push()`方法返回新数组的长度
`pop()`方法返回被移除的元素
`unshift()`, `shift()`对应`push()`, `pop()`，但是对数组头进行操作
`push()`和`unshift()`可以加入多个元素，用`,`分隔
`splice(startIndex, numToRemove, ...itemToAdd)`在原数组上进行操作
`splice(startIndex)`删除`startIndex`后的元素
`splice()`将返回数组的浅拷贝副本
`includes(valueToSearch, startIndex)`
`concat(arr)`也可以拼接数组内容

关于数组解构
```js
let numbers = [1, 2, 3];
let [first1, second1, third1] = numbers; // 获取数组元素
let [first2, , third2] = numbers; // 跳过第二个元素
let [first3, second3, third3, fourth = 4] = numbers; // 缺省值
let [first4, ...rest] = numbers; // 捕获剩余元素为新数组
// ...rest必须为数组解构的最后一个元素
```

反转字符串：
`string`是不可变的，必须新建一个字符串以代替旧的，这意味着将字符串转为数组再进行反转会更高效
