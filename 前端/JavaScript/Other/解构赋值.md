JS中最常用的数据结构就是 `Object` 和 `Array`。
解构对象用`{}`，数组用`[]`，数组解构从“列表”里取值，看顺序；对象解构从“字典”里取值，看属性名。

数组项无法使用重命名`:`

对象解构
```js
let options = {
  title: "Menu"
};

let {width: w = 100, height: h = 200, title} = options;

alert(title);  // Menu
alert(w);      // 100
alert(h);      // 200
```

### 不使用let时的陷阱

如果不在声明变量时进行解构，`{...}`会被视为代码块而不是解构赋值，这时需要用`()`将语句包裹起来
```js
let title, width, height;
({title, width, height} = {title: "Menu", width: 200, height: 100});
alert( title ); // Menu
```


### 智能函数参数

JS是动态语言，不像C++/Java可以通过编译期检查函数重载来匹配不同的参数列表。如果使用参数列表传参需要使用`undefined`占位
```js
function showMenu(title = "Untitled", width = 200, items = []) {
  // ...
}

// 在采用默认值就可以的位置设置 undefined
showMenu("My Menu", undefined, ["Item1", "Item2"])
```
