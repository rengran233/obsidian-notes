HTML骨架
父子嵌套/兄弟并列
标题标签`<h1>-<h6>` h1只能有一个，各标题不能嵌套
文本格式化标签
图像标签`<img src alt >`
语义化标签：标签的默认效果不重要，语义最重要
块级元素和行内元素
块级内可以有块级和行内，行内里只能有行内
p标签内不能有块级元素
包装：div块，span行，自身无效果
换行会被解析为**一个**空格
a是行内元素，但是可以包裹除自身外的任何元素
全局属性id, class, dir, title, lang
id不能有空格，且不能重复；class都可以，空格可以用于分割多个类名，class规范是用-连接单词
figure：有意义的媒体内容



优先级：行内>内部=外部
```css
h1 /* 选择器 */ {
	color: red;                  |  声明块
	font-size: 12px;   | 声明     |
}
```
通配选择器：清除样式，`*`
元素选择器：`<元素名>`
类选择器：最常用，`.<类名>`
ID选择器：主要用于JS交互和锚点链接，`#<ID>`
交集选择器：从大到小，多用元素配合类名，中间没有空格
并集选择器：用`,`分隔
后代选择器：用空格分隔
子代选择器：用`>`分隔
兄弟选择器：用`+`分隔
属性选择器：用`[]`包裹

伪类选择器：`:伪类名` link, visited, hover, active

`<meta name="description" content="">` 给SE看的描述，content中是内容
`<meta property="" content="">` 用OG协议控制社交媒体显示



模板字符串（template literals）

闭包：高阶函数`f1`返回的函数`f2`需要用到`f1`中的变量`a`，于是`f1`执行后`a`也不会被销毁，此时只有`f2`能操作`a`

```ts
let person: {
  name: string
  age: number
}

person = {
  name: 'John',
  age: 25,
}

interface User {
  name: string
  age: number
}

let user: User = {
  name: 'Alice',
  age: 30,
}
```

在react中遍历数组使用`.map()`方法，如果需要生成大量相似标签，需要将每一项的编号（最好用每一项专有的id）作为第二个参数输入以帮助react高效处理列表的变化

js的short-circuit规则和python一样，都是返回导致逻辑运算结束的项，但是js中空数组`[]`和空对象`{}`都是真值

jsx中的内联样式的属性名称要用驼峰而非蛇形表示，同时用两层`{{}}`包裹，内层表示js对象，外层为jsx语法

给组件传参不必传整个对象，可以用形如`{<member1>, <member2>}`将对象的成员变量解构出来，对应的类型推断为`{<member1>}: <object>`或者`{<member1>}: {<member1>: <type>}`
对应的，主动将一个对象解构需要用`...`展开操作符
