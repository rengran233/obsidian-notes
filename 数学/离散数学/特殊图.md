# 📚 图论深度笔记：遍历、匹配与平面性

## 一、 欧拉图 (Eulerian Graphs) —— 边的遍历
**核心定义：** 经过图中每条边恰好一次。

### 1. 判定定理
*   **无向图 $G$：**
    *   **欧拉回路**：$G$ 连通且所有顶点的度数均为**偶数**。
    *   **欧拉通路**：$G$ 连通且恰有 **0 个或 2 个**奇度顶点（若有 2 个，则它们必为起点和终点）。
*   **有向图 $D$：**
    *   **欧拉回路**：$D$ 强连通且每个顶点的入度等于出度 ($d^-(v) = d^+(v)$)。
    *   **欧拉通路**：$D$ 弱连通，且除起点（出度-入度=1）和终点（入度-出度=1）外，其余点入度等于出度。

### 2. Fleury (弗罗里) 算法
**思想：** “除非别无选择，否则不走‘桥’”。
*   **步骤：** 任意选择起点，每次选择边时，若该边不是当前剩余图的割边（桥），则优先选它；若是割边，则仅在没有其他边可选时才走。

> **示例：** 柯尼斯堡七桥问题。由于 4 个顶点的度数均为奇数（3,3,3,5），违反了判定定理，故不存在欧拉通路。

## 二、 哈密顿图 (Hamiltonian Graphs) —— 点的遍历
**核心定义：** 经过图中每个顶点恰好一次。

### 1. 判定条件
*   **必要条件**：若 $G$ 是哈密顿图，则对于 $V$ 的任何非空真子集 $S$，有 $\omega(G-S) \le |S|$（其中 $\omega$ 为连通分支数）。
    *   **示例**：若删掉 2 个点后图变成了 3 个孤立部分，则该图必不是哈密顿图。
*   **充分条件 (Ore 定理)**：对于 $n \ge 3$ 的简单图，若任意不相邻的一对顶点 $u, v$ 都有 $d(u) + d(v) \ge n$，则 $G$ 是哈密顿图。
*   **推论 (Dirac 定理)**：对于 $n \ge 3$ 的简单图，若每个顶点的度数 $d(v) \ge n/2$，则 $G$ 是哈密顿图。

### 2. 判定方法（发散）
*   **闭包法 (Bondy-Chvátal)**：反复连接度数和 $\ge n$ 的不相邻点，最终若得到完全图，则原图是哈密顿图。
*   **二部图判定**：若二部图 $G(X, Y)$ 中 $|X| \neq |Y|$，则必不存在哈密顿回路。

## 三、 二部图 (Bipartite Graphs) 与匹配

### 1. 偶图 (二部图)
*   **定义**：顶点集可划分为 $V_1, V_2$，使得边只存在于两集之间。
*   **充要条件**：图 $G$ 是二部图 $\iff$ **$G$ 中不含长度为奇数的回路**。

### 2. 匹配与霍尔定理 (Hall's Theorem)
*   **匹配**：边集 $M$ 中任意两条边不共用端点。
*   **相异性条件 (Hall 条件)**：二部图 $G(X, Y)$ 存在饱和 $X$ 的匹配，当且仅当对于 $X$ 的任意子集 $S$，都有 $|N(S)| \ge |S|$（$N(S)$ 是 $S$ 的邻点集）。
*   **t-条件 (扩展)**：若每个 $X$ 节点的度至少为 $t$，每个 $Y$ 节点的度至多为 $t$，则必存在饱和 $X$ 的匹配。

> **示例应用**：5 名工人申请 5 项工作。若任意 $k$ 名工人总共能胜任的工作种类少于 $k$，则根据霍尔定理，无法实现每人分配一项工作。

## 四、 平面图 (Planar Graphs)

### 1. 基本概念
*   **平面图**：可以在平面上绘制且边不相交（除了在顶点处）。
*   **面 (Face)**：由边围成的区域。**次数 (Degree, $deg(f)$)**：围成面的边界边数（割边算两次）。
*   **重要性质**：所有面的次数之和等于边数 $m$ 的两倍：$\sum deg(f_i) = 2m$。

### 2. 欧拉公式 (Euler's Formula)
*   **公式**：对于连通平面图，$n - m + f = 2$。
*   **推论 1**：若 $G$ 是 $n \ge 3$ 的简单连通平面图，则 $m \le 3n - 6$。
*   **推论 2**：若 $G$ 且每个面的次数 $\ge k$，则 $m \le \frac{k(n-2)}{k-2}$。

> **示例：$K_5$ 为何非平面图？**
> $n=5, m=10$。代入 $m \le 3n-6$，得 $10 \le 3(5)-6 = 9$，矛盾！故 $K_5$ 不是平面图。
> **示例：$K_{3,3}$ 为何非平面图？**
> 二部图无奇圈，故最小面次数 $k=4$。代入 $m \le 2n-4$，得 $9 \le 2(6)-4 = 8$，矛盾！

### 3. 同胚与库拉托夫斯基定理
*   **2 度结点收缩/扩充**：在边上增加或删除 2 度结点，不改变图的拓扑本质。
*   **同胚 (Homeomorphic)**：两个图可以通过对 2 度结点的一系列操作互相转化。
*   **库拉托夫斯基定理 (Kuratowski's Theorem)**：
    图 $G$ 是平面图 $\iff$ **$G$ 不包含与 $K_5$ 或 $K_{3,3}$ 同胚的子图**。
    *   *意义*：这是判定平面性的终极武器，指出了非平面图的最小“核心”。

### 📝 总结表：欧拉图 vs 哈密顿图

| 特性       | 欧拉图 (Euler)    | 哈密顿图 (Hamilton)  |     |                  |
| :------- | :------------- | :--------------- | --- | ---------------- |
| **关注点**  | 边 (Every Edge) | 点 (Every Vertex) |     |                  |
| **判定难度** | 容易 (看度数即可)     | 困难 (NP-Complete) |     |                  |
| **必要条件** | 连通，度数为偶数       | $                | S   | \ge \omega(G-S)$ |
| **应用场景** | 邮递员路线、扫路车      | 旅行商问题 (TSP)、物流配送 |     |                  |
