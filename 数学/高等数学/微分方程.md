微分方程是高等数学中题型最固定、套路最清晰的章节之一。其解题核心在于：**识别类型 $\rightarrow$ 套用公式 $\rightarrow$ 计算积分**。

以下是微分方程所有常考题型的全维度总结。

---

### 第一部分：一阶微分方程（识别与直接求解）

这是考试的基础，重点在于方程的变形与识别。

#### 1. 可分离变量方程
*   **特征**：$y' = f(x)g(y)$。
*   **方法**：将所有 $y$ 移到 $dy$ 边，所有 $x$ 移到 $dx$ 边，两边积分。

#### 2. 齐次方程
*   **特征**：$y' = \phi(\frac{y}{x})$。
*   **方法**：令 $u = \frac{y}{x}$，则 $y = ux$，$y' = u + x u'$。代入原方程化为可分离变量方程。

#### 3. 一阶线性微分方程（核心考点）
*   **特征**：$y' + P(x)y = Q(x)$。
*   **方法**：直接套用**通解公式**（见最后公式汇总）。

#### 4. 伯努利方程
*   **特征**：$y' + P(x)y = Q(x)y^n \quad (n \neq 0, 1)$。
*   **方法**：两边同除以 $y^n$，令 $z = y^{1-n}$，转化为关于 $z$ 的一阶线性方程。

---

### 第二部分：高阶微分方程（降阶与线性）

#### 5. 可降阶的高阶方程
*   **$y^{(n)} = f(x)$ 型**：连续积分 $n$ 次。
*   **$f(x, y', y'') = 0$ 型（缺 $y$）**：令 $p = y'$，则 $y'' = p'$，降为一阶。
*   **$f(y, y', y'') = 0$ 型（缺 $x$）**：令 $p = y'$，则 $y'' = p \frac{dp}{dy}$，降为一阶。

#### 6. 高阶常系数齐次线性方程
*   **特征**：$y'' + py' + qy = 0$。
*   **方法**：写出特征方程 $r^2 + pr + q = 0$，根据根的情况直接写解。

#### 7. 高阶常系数非齐次线性方程（计算难点）
*   **特征**：$y'' + py' + qy = f(x)$。
*   **方法**：**通解 = 齐次通解 + 非齐次特解**。
*   **特解求法**：根据 $f(x)$ 的形式使用**待定系数法**。

---

### 第三部分：综合应用题型

#### 8. 几何应用题
*   **特征**：给出切线、法线、截距、面积等几何性质。
*   **方法**：根据几何公式列出微分方程。例如：切线斜率 $y' = \tan \alpha$。

#### 9. 物理应用题
*   **特征**：变加速运动、人口增长、冷却定律、浓度变化。
*   **方法**：利用牛顿第二定律（$F=ma$）或相关物理定律建模。

#### 10. 变限积分微分方程
*   **特征**：方程中含有 $\int_a^x f(t)dt$。
*   **方法**：利用变限积分求导公式 $\frac{d}{dx}\int_a^x f(t)dt = f(x)$，两边同时求导，转化为常规微分方程。

---

### 第四部分：核心公式汇总

#### 1. 一阶线性通解公式
对于 $y' + P(x)y = Q(x)$：
$$y = e^{-\int P(x)dx} \left[ \int Q(x) e^{\int P(x)dx} dx + C \right]$$

#### 2. 特征方程与齐次解（二阶为例）
对于 $y'' + py' + qy = 0$，特征方程 $r^2 + pr + q = 0$：
*   **$r_1, r_2$ 为不等实根**：$y = C_1 e^{r_1 x} + C_2 e^{r_2 x}$
*   **$r_1 = r_2 = r$ 为重根**：$y = (C_1 + C_2 x) e^{rx}$
*   **$r = \alpha \pm \beta i$ 为复根**：$y = e^{\alpha x} (C_1 \cos \beta x + C_2 \sin \beta x)$

#### 3. 非齐次特解 $y^*$ 的设法（待定系数法）
对于 $y'' + py' + qy = f(x)$：
*   **若 $f(x) = P_n(x)e^{\lambda x}$**：
    设 $y^* = x^k Q_n(x)e^{\lambda x}$
    （$k$ 的取值：$\lambda$ 不是特征根 $k=0$；$\lambda$ 是单根 $k=1$；$\lambda$ 是重根 $k=2$）
*   **若 $f(x) = e^{\alpha x}[P_l(x)\cos\beta x + Q_n(x)\sin\beta x]$**：
    设 $y^* = x^k e^{\alpha x} [A_m(x)\cos\beta x + B_m(x)\sin\beta x]$，其中 $m = \max\{l, n\}$。
    （$k$ 的取值：$\alpha + \beta i$ 不是特征根 $k=0$；是特征根 $k=1$）

#### 4. 欧拉方程（数一重点）
对于 $x^2 y'' + pxy' + qy = f(x)$：
*   **变换**：令 $x = e^t$（或 $t = \ln x$），则 $xy' = \frac{dy}{dt}$，$x^2 y'' = \frac{d^2 y}{dt^2} - \frac{dy}{dt}$。

---

### 💡 复习建议
1.  **积分是命脉**：微分方程解不出来，通常不是方程没认准，而是基本积分公式不熟。
2.  **常数 $C$ 别丢**：每积一次分必须加一个 $C$，二阶方程最后必须有两个独立的常数 $C_1, C_2$。
3.  **注意初值**：如果有 $y(0)=a$ 等条件，求出通解后一定要代入求出确定的 $C$。
