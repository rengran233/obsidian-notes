线性关系

```cpp
struct ListNode 
{ 
	int val; 
	ListNode* next; 
	ListNode(int x) : val(x), next(nullptr) {} 
};
```

head         n1         n2                  `head->next = n1`
   |          /    |       /    |                   `n1->next = n2`
next(A)    next(B)    next(C)  
若令`head->next = head->next->next`，由A-B-C变为A-C
head          n2
   |          /     |
next(A)      next(C)                  删去n1节点
