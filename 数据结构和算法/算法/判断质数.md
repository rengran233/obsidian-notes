## 试除法：逐个判断
```cpp
for (int i = 2; i <= x / i; ++i) // i * i <= x即i <= sqrt(x)
	if (x % i == 0) return 0;
return 1;
```

## 埃氏筛：删除合数
```cpp
for (int i = 2; i <= N / i; ++i)
	if (!pri[i])
		for (int j = i * i; j <= N; j += i) pri[j] = 1;
```
只筛去了合数，i截止sqrt(N)，如果需要统计质数需要外加一层循环


## 欧拉筛：不重复删除
```cpp
for (int i = 2; i <= x; ++i)
{
	if (!st[i]) primes[++pp] = i;
	for (int j = 1; primes[j] < x / i; ++j)
	{
		st[primes[j] * i] = 1;
		if (primes[j] % x == 0) break;
	}
}
```
