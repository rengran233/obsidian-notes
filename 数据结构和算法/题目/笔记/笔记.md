对于`[left, right]`二分搜索使用整数除法计算中间指针`mid`，结果小于等于真实值，如果简单赋值`right = mid - 1`可能使右指针小于左指针，所以循环条件必须使用`left < right`，同时因为左右指针相等时也有可能指向目标值，循环条件应为`left <= right`
建议使用`[left, right)`区间

左闭右开区间在初始状态`[0, 0)`时不含元素，有利于处理边界条件

`multiset.erase(key)` will erases all the key elements in the multiset

while (a) {
