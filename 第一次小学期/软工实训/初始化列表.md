```cpp
Bullet::Bullet(GameObject::ObjectType ownerType, int damage, float speed, QObject* parent)
    : MovableObject(GameObject::ObjectType::Bullet, parent), m_damage(damage), m_ownerType(ownerType) // 调用父类构造函数并初始化成员变量
{
    setBaseSpeed(speed); // 设置子弹基础移动速度
}
```
有基类的类的构造函数的初始化列表中要包含基类构造

- 子类初始化成员变量时，只能初始化自己声明的成员，父类成员要通过父类构造函数。
