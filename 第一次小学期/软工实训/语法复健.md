`int getDamage() const;`

`void update(float deltaTime) override;`
头文件中函数定义后如果有修饰符也要写出来


在派生类中调用父类的构造函数
```cpp
class A {
public:
    A(int x, int y) : m_x(x), m_y(y) {}
    // ...
protected:
    int m_x;
    int m_y;
};

class B : public A {
public:
    B(int x, int y, int z) : A(x, y), m_z(z) {}
    // ...
private:
    int m_z;
};
```
- 如果你没有在子类构造函数的初始化列表中显式调用父类构造函数，编译器会自动调用父类的无参构造函数（如果有）。
 - 如果父类没有无参构造函数，你必须在子类构造函数的初始化列表中显式调用父类的有参构造函数，否则编译不通过。

- 构造函数的参数默认值应该在头文件（定义时）给出
- 在头文件给出后，源文件不应也不能再次给出
- 如果父类已经给出默认值，则不应该在派生类中再次声明
